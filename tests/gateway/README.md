## Purpose

Tests for the main gateway server routing.

## Files

| File | Purpose |
|------|---------|
| `test-routing.js` | Route mounting and request handling tests |

## Running

```bash
node tests/gateway/test-routing.js
```

- SDK route mounting

- Health endpoints
- SSE event streaming
